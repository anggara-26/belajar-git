Mengapa Harus Belajar Git & GitHub?

Karena Git & GitHub adalah tools yang sangat powerful untuk mengelola kode program. Dengan Git & GitHub, kita bisa bekerja secara kolaboratif dengan orang lain. Selain itu, Git & GitHub juga memudahkan kita dalam mengelola versi dari kode program yang kita buat.
